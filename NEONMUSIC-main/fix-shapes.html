🔧 Fix Shapes Settings

Jika shapes hilang atau bermasalah, gunakan tools ini untuk memperbaiki.

Troubleshooting:

  • Shapes hilang setelah ubah speed:
    Klik "Clear Shapes Settings" lalu refresh page
  • Shapes tidak muncul sama sekali:
    Klik "Reset to Defaults" lalu refresh page
  • Settings tidak tersimpan:
    Check browser localStorage enabled
  • Animasi rusak:
    Clear settings dan refresh page 2x

Manual Fix:

Buka browser console (F12) dan jalankan:

localStorage.removeItem('neonwave_shapes_settings')

Lalu refresh page.
