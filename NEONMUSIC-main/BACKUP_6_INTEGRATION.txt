                    FILE 1: spotify-config.js
================================================================================

LOKASI: /spotify-config.js
DESKRIPSI: Konfigurasi credentials untuk Spotify Web API

STRUKTUR:
```javascript
const SPOTIFY_CONFIG = {
  clientId: "YOUR_CLIENT_ID",
  clientSecret: "YOUR_CLIENT_SECRET",
  redirectUri: "http://localhost:8000/callback",
  scopes: [
    "user-read-private",
    "user-read-email",
    "user-library-read",
    "user-library-modify",
    "playlist-read-private",
    "playlist-modify-public",
    "playlist-modify-private",
    "user-top-read",
    "user-read-recently-played",
    "streaming"
  ]
};
```

KONFIGURASI:
1. Client ID
   - Didapat dari Spotify Developer Dashboard
   - Format: 32 karakter alphanumeric
   - Public identifier

2. Client Secret
   - Didapat dari Spotify Developer Dashboard
   - Format: 32 karakter alphanumeric
   - RAHASIA - jangan expose di frontend

3. Redirect URI
   - URL callback setelah authorization
   - Harus didaftarkan di Spotify Dashboard
   - Contoh: http://localhost:8000/callback

4. Scopes (Permissions)
   - user-read-private: Baca profil user
   - user-read-email: Baca email user
   - user-library-read: Baca library user
   - user-library-modify: Modifikasi library user
   - playlist-read-private: Baca playlist private
   - playlist-modify-public: Edit playlist public
   - playlist-modify-private: Edit playlist private
   - user-top-read: Baca top tracks/artists
   - user-read-recently-played: Baca recently played
   - streaming: Streaming playback

CARA MENDAPATKAN CREDENTIALS:
1. Buka https://developer.spotify.com/dashboard
2. Login dengan akun Spotify
3. Klik "Create an App"
4. Isi nama dan deskripsi app
5. Copy Client ID dan Client Secret
6. Tambahkan Redirect URI di settings
7. Save

ERROR HANDLING:
```javascript
try {
  const tracks = await getSavedTracks();
  renderTracks(tracks);
} catch (error) {
  if (error.status === 401) {
    // Token expired, refresh
    await refreshAccessToken();
    // Retry
    const tracks = await getSavedTracks();
    renderTracks(tracks);
  } else {
    showToast("Error loading tracks", "error");
    console.error(error);
  }
}
```

RATE LIMITING:
Spotify API has rate limits:
- 429 status code = rate limited
- Retry-After header = seconds to wait
- Implement exponential backoff

```javascript
async function apiRequest(url, options, retries = 3) {
  try {
    const response = await fetch(url, options);
    
    if (response.status === 429) {
      const retryAfter = response.headers.get('Retry-After');
      await sleep(retryAfter * 1000);
      return apiRequest(url, options, retries - 1);
    }
    
    return response.json();
  } catch (error) {
    if (retries > 0) {
      await sleep(1000);
      return apiRequest(url, options, retries - 1);
    }
    throw error;
  }
}
```
