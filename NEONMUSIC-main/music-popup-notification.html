🎵 Music Popup Notification

Premium dark futuristic glassmorphism popup component for music apps. Click buttons below to test.

📝 Usage Example:

// Simple usage
showMusicPopup(
    'Timeless',           // title
    'The Weeknd',         // artist
    'images/cover.jpg',   // cover URL
    4000                  // duration (optional, default 4000ms)
);

// Or use the class directly
window.musicPopup.show('Song', 'Artist', 'cover.jpg', 5000);

// Smart notification (popup + browser notification if tab inactive)
window.musicPopup.smartNotify('Song', 'Artist', 'cover.jpg');

🎨 Features:

✅ Dark glassmorphism design
✅ Multi-layer neon glow
✅ Smooth animations (cubic-bezier)
✅ Auto-hide after 4 seconds
✅ Smart replace (no spam)
✅ Mini progress bar
✅ Hover effects
✅ Responsive design
✅ Browser notification support
✅ No external dependencies
